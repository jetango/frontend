# Greg Common Application

The start is here.

# Instalation

1. Clone
2. Run `./init.sh` or copy `settings.example.json` to `settings.json`
3. Edit `settings.json`
4. Run `gulp` or  `gulp <command>`

# Commands:

### Build
1. `build:html`
2. `build:css`
3. `build:js`
4. `build:img`
5. `build:libs`
6. `build:clean`
7. `build:watch`
8. `build`

### Release
1. `release:css`
2. `release:js`
3. `release:img`
4. `release:libs`
5. `release:clean`
6. `release:watch`
7. `release`
